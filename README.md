# Description

This is a simple Market Equilibrium Visualiser intended to allow Commerce students studying in 12th Grade (CBSE) to understand the relationship between supply, demand and price using an interactive graph application. This visualiser allows them to grasp
the concept better and is tailored to prepare them for their Board exams.

# Tools used
- Language: Python
- Packages used: Streamlit, Numpy, Matplotlib
